
# S-STAGE Website

A Vite + React + Tailwind project ready for GitHub and Vercel deployment.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```

## Deploy to Vercel

1. Upload this whole folder to GitHub.
2. Import the repository in Vercel.
3. Build command: `npm run build`
4. Output directory: `dist`

Vercel usually detects Vite automatically.
